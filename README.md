# soon

## Deskripsi
project ini dibuat untuk mengatasi masalah yang terjadi di beberapa sekolah yaitu ketika guru/staff/yang berkepentingan ingin memanggil siswa dan juga saat ada pemberitahuan melalui pengeras suara. Dimana sering kali suara panggilan tersebut tidak cukup jelas yang menyebabkan kebingungan serta berhentinya pembelajaran seluruh sekolah sesaat adanya pemberitahuan.

## Pengguna
- Guru / staff TU
- display disetiap kelas

## cara kerja
- pada interface guru tersedia pilihan kelas X, XI, XII dan pilihan kelas A hingga K
- setelah memilih kelas akan ada tempat mengisi keterangan panggilan dsb
- akan muncul nama serta keterangan yang dimasukkan guru pada display kelas

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
