# Realtime Student Caller

## Features

- School officer/teacher/headmaster can send a realtime message to each client
- Client to listen to any new messages

## Technologies

- Backend:
  - Node JS
  - Websocket
- Frontend:
  - HTML
  - Vanilla JS
