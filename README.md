# Realtime Student Caller

## Features

- One way chat. so only headmaster or teacher can send message from office.
- Minimizing distractions during learning.
- The message is delivered clearly and in real time.

## Technologies

- Backend:
  - Node JS
  - Websocket
- Frontend:
  - HTML
  - Vanilla JS
  - CSS
